- 今まで学んできた内容を応用して自社の業務改善に繋がるアプリを作ってみた。
- レーダーチャートで診断が出るようにしてみた

## ⑥難しかった点・次回トライしたいこと(又は機能)

- まだ応用レベルも最低限の部分のため、動的な動きを付けたり、UI/UXにこだわりきれていない
- 時間があったら、もう少し挙動を付けたり、商談の際に使いやすいように、ブラッシュアップしてみたい
- チェック部分からの診断の紐付けは、まだ完成値に届いていないので、自社で活用できるレベルではない
